Instructions:

Use the arrow keys on your keyboard to move around the cube.
Your goal is to collect all the coins in the maze. As you collect coins, your player cube will grow in size.
A little bit of info about the player such as the size and score are displayed to the right of the game.
Note: The player starts out quite small so look carefully for the cube.
Have fun!

The Maze Game

Notes about the project: This project was coded using a library called p5.js which streamlines the use of the html canvas into a more user friendly process. In order to generate the maze, the program uses a Depth First Search (DFS) traversal.
